# github技巧卡包 #

----------

## **1. github是什么？** ##

- 它是全世界最优秀程序员汇聚的开源社区
- 它是国际著名代码托管平台
- 它代表了全世界编程领域的趋势

## **2. 起航** ##

- 对于36岁的我，从0开始学习可能只有程序员才会碰的github，算是一个不大不小的挑战，也是刷新我认知边界的一次旅行吧！
- 既然已经决定，多难也得从最小行动开始，从卡片起航，Let’s Go ！
